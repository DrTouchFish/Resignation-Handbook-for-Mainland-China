# 举证
* 合同协议：带公司印章的收入证明、劳动合同、劳动合同变更协议等
* 录音录像：录像最好录入人脸，录音要在录音内容明确对方身份，一定要保持连贯没有中断
* 其他证据：使用简证app（杭州市互联网公证处主导下开发，可以做付费公正，也可以线上公正后去线下取公正单）采集并公证相关录屏和照片未被篡改（防止对方提出质证，证明有效性）

# 公积金篇
基于现行浙江省《住房公积金管理条例》相关固定浙江省的公司必须给员工全额缴纳公积金，缴存比例不得低于上一年度月平均（含年终、奖金）工资的5%。
需固定证据：公司盖章的月税前工资
追缴方式：杭州市公积金管理中心线下办理

# 社保篇（玉石俱焚）
抱着这钱公司不给就谁都别想要，为伟大祖国做贡献。去税务举报公司未足额缴纳公积金，社保欠缴会产生滞纳金，按日计息，像高利贷一样抽筋剥皮，一告一个准，用脚都能赢。

# 防止诱导你放弃权利
依据《中华人民共和国劳动合同法》第二十六条规定劳动合同里不能要求你放弃合法权利
公司如果要和你签放弃追缴权利的协议是无效的，依据《民法典》 第四百九十七条 与 第一编第六章第三节中第一百五十三条规定 违反法律、行政法规的强制性规定的民事法律行为无效
但如果协议内容里写你同意放弃权利后公司给你多少多少钱，如果你去追缴公司可以告你要回这钱，但你可以和公司拖到一审二审再到强制执行都行，公司钱也花不少。

# 谈赔偿
只要你的开价 ＜ 追缴公积金总额 + 社保追缴总额 + 社保欠缴产生的滞纳金
一般他们都会答应你的诉求，别管他们说什么，都只是纸老虎罢了，不给就在职追缴社保公积金，全额社保公积金上班不算亏。
不要当心背调，背调电话是你自己填的。

# 抛弃幻想、准备斗争
思路：集团的相关规定就算经过全体职工意见采集、职工大会审议通过并执行，但集团得举证参会的职工代表是由民主程序选举产生的，同时具有代表性，人员组成在《工会法》都有要求。而且作为具有独立民事责任的各个子公司不能直接适用集团的规定，需要单独再走一遍民主程序（民主选举工会或举行全体员工大会）推行条款。条款本身合法性不成立，所以处罚非法。公司无权扣你工资，最多只能扣年终等绩效奖金。

参考案例：山东省日照市中级人民法院相关判例

相关条例（相关条款出处与原文）：
- 《中华人民共和国公司法》第十三条规定，子公司具有企业法人资格，依法独立承担民事责任。
- 《浙江省实施〈中华人民共和国工会法〉办法》 第六条  用人单位应当在开业或者设立一年内依法建立工会组织。民主选举产生工会委员会
- 《中华人民共和国劳动法》 第八条　劳动者依照法律规定，通过职工大会、职工代表大会或者其他形式，参与民主管理或者就保护劳动者合法权益与用人单位进行平等协商。
- 《中华人民共和国劳动合同法》 第四十三条 用人单位单方解除劳动合同，应当事先将理由通知工会。用人单位违反法律、行政法规规定或者劳动合同约定的，工会有权要求用人单位纠正。用人单位应当研究工会的意见，并将处理结果书面通知工会。
- 《中华人民共和国劳动合同法》 第四十七条 经济补偿按劳动者在本单位工作的年限，每满一年支付一个月工资的标准向劳动者支付。六个月以上不满一年的，按一年计算；不满六个月的，向劳动者支付半个月工资的经济补偿。
- 《中华人民共和国劳动合同法》 第五十一条 企业职工一方与用人单位通过平等协商，可以就劳动报酬、工作时间、休息休假、劳动安全卫生、保险福利等事项订立集体合同。集体合同草案应当提交职工代表大会或者全体职工讨论通过。集体合同由工会代表企业职工一方与用人单位订立；尚未建立工会的用人单位，由上级工会指导劳动者推举的代表与用人单位订立。
